Welcome

The plugin sumdate is a utilitare for dates manipulations. Converting date type for String to Date or Date to String.

Required

jQuery 1.6.0 or more

Include plugin

<head>
    <meta charset="utf-8" />
    <script type="text/javascript" src="jquery-1.8.3.js"></script>
    <script type="text/javascript" src="jquery.sumdate.min-0.2b.js"></script>
</head>

Use direct mode, pass date value in a parameter "date" Ex:

var date =new Date();
date.setMonth(1); // February
date.setDate(27);
var newDate = $.sumdate({methods:['sum'],"date":date,"sumValues":{"days":5}});
alert(newDate); // Mon Mar 04 2013 15:38:28 GMT-0300 (Hora oficial do Brasil)

Use selector mode, pass the HTML element that contains a date value

Ex:

<div id="divDate">20/12/2013</div>
<script type="text/javascript">
    var newDate = $('#divDate').sumdate({
        methods:['sum'],
        "sumValues":{"days":5},
        pattern:'DD/MM/YYYY'});
    alert(newDate); //25/12/2013
</script>
