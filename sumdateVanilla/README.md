Sumdate
=========

A small library that adds and validate dates

## Installation

  `npm install @lucianomprado/sumdate`

## Usage

    var SumDate = require('@lucianomprado/sumdate');

    var formattedNum = new SumDate('03/01/1983');


## Tests

  `npm test`
